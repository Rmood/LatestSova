Stack Overflow Viewer Application

About

The purpose of this application was to give an idea of how programmers could browse through a static database based on
Stack Overflow and find relevant Q/As based on their search by using specific keywords.

How it works

The way our application is supposed to function is, a user can insert certain keywords, in order to receive
Q/As related in computer programming.
